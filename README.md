# Banco Digital POO

Projeto simples de um banco digital desenvolvido em Java utilizando os princípios de Programação Orientada a Objetos (POO).

## 📚 Tecnologias e Conceitos

- Java
- Programação Orientada a Objetos (POO)
- Herança, Abstração e Encapsulamento

## 📁 Estrutura

O projeto simula operações básicas bancárias com as seguintes classes principais:

- `Banco`: Armazena informações sobre o banco e suas contas.
- `Cliente`: Representa um cliente do banco.
- `Conta`: Classe abstrata com métodos comuns.
- `ContaCorrente` e `ContaPoupanca`: Extendem `Conta` e implementam regras específicas.
- `Main`: Classe principal que executa o programa.
